---
title: "Avatar video formats accepted"
description: "Description of your new file."
---

* Formats accepted: .mov and .mp4

* Dimensions: 16:9 or 9:16&#x20;

* Accepted resolutions:&#x20;
  width: 3840, height: 2160, // 4K
  width: 2160, height: 3840, // 4K portrait
  width: 2560, height: 1440, // 2K
  width: 1440, height: 2560, // 2K portrait
  width: 1920, height: 1080, // 1080p
  width: 1080, height: 1920, // 1080p portrait
  width: 1280, height: 720, // 720p
  width: 720, height: 1280, // 720p portrait

* Max size: 5GB

<Info>
  **Struggling to convert your file?**&#x20;
  Using a free software like CapCut, you will be able to resize and reformat your video as you wish.&#x20;
</Info>
